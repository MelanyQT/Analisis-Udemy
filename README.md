# 📊 Análisis de Mercado y Predicción: Cursos de Data Science en Udemy

Este proyecto utiliza **Python** y **Machine Learning** para analizar el ecosistema de cursos de análisis de datos en Udemy, identificando patrones de éxito y construyendo un modelo predictivo para la captación de estudiantes.

## 🚀 Ver Dashboard en Vivo
Puedes interactuar con los datos y probar el modelo predictivo aquí: 
👉 https://melanynatalia.streamlit.app/


## 📝 Resumen del Proyecto
El análisis se basó en un dataset de **3,443 cursos**, enfrentando retos de limpieza de datos internacionales (idioma ruso) y normalización de métricas de engagement.

### Hallazgos Clave:
* **Engagement:** El rating promedio es de **4.33/5.0**, lo que indica una alta satisfacción en la categoría.
* **Mercado:** El precio mediano se sitúa en **$5,390**.
* **Modelo Predictivo:** Se entrenó una Regresión Lineal con un coeficiente de **5.44**. 
    * *Interpretación:* Por cada nueva reseña, se estima la llegada de aproximadamente 5 estudiantes adicionales.

## 🛠️ Tecnologías Utilizadas
* **Análisis de Datos:** Pandas, NumPy.
* **Visualización:** Matplotlib (Gridspec).
* **Machine Learning:** Scikit-Learn (Linear Regression).
* **Despliegue:** Streamlit Cloud.
